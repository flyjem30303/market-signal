# Stock Page Strategy

CEO 對股票頁的定位：股票頁不是只有健康度與回檔風險燈號，而是「股市資料頁 + 智能解讀頁」。

## 產品分層

### 基礎行情層

使用者期待每個股票頁都能看到：

- 股票名稱與代號。
- 收盤價、漲跌、漲跌幅。
- 成交量。
- 本益比、本淨比、殖利率。
- 開盤、最高、最低。
- 新聞與基本資料。

這一層不是差異化，但缺少會降低信任感。

### 智能解讀層

本網站的差異化：

- 多頭健康度。
- 回檔風險度。
- 資料品質。
- 新聞信心評論。
- 回測摘要。
- 晨報與週報把資料轉成投資節奏提醒。

## 不要做成大型股市網站複製品

我們不靠資料量贏大型股市網站，而是靠資料整理後的風險判讀贏。
