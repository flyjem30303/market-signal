PM may open the executable packet window only when one of these is true:

| Condition | Status | PM action |
| --- | --- | --- |
| Hosting project name is known | `platform_generated_value_required` | Record plain project name only, no dashboard token or private URL. |
| Temporary Beta URL is known | `platform_generated_value_required` | Record public URL only, no secret query string or invite token. |
