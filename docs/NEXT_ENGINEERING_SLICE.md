# 下一個工程切片

## 目標

把目前 Next.js 骨架跑起來，並將靜態原型的第一頁「標的選擇 + 今日燈號」完整搬進 React。

## 為什麼先做這個

- 它是所有股票頁的核心。
- 它不需要先接真實資料庫。
- 它能驗證 Next.js 架構與 SEO route。
- 它之後可以直接接 Supabase。

## 任務拆解

### 1. 安裝依賴

```bash
npm install
```

目前已安裝過，若換電腦再重新執行即可。

### 2. 啟動網站

```bash
npm run dev
```

若 Codex 沙盒內出現 `EPERM lstat C:\Users\C0298`，請在一般 PowerShell 視窗中執行。

### 3. 檢查頁面

```text
/
/stocks/2330
/stocks/0050
/weekly
```

### 4. 搬移功能

- 標的搜尋
- 愛心收藏 localStorage
- 今日健康度/風險度雙圓環
- 六大模組
- 健康度 × 風險度象限

### 5. 暫時不做

- 真實資料更新
- 會員登入
- 廣告
- 聯盟行銷
- 新聞 API
