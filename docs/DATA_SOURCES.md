# 資料來源草案

## 官方資料

- TWSE OpenAPI：上市股票、每日交易、指數、估值、三大法人。
- TPEx OpenAPI：上櫃股票、每日交易、法人與融資融券。
- 公開資訊觀測站：財報、營收、重大訊息。
- 期交所：台指期、外資期貨部位。
- 中央銀行：匯率。

## 國際資料

- FRED：美國 10 年期公債殖利率、利率與宏觀資料。
- Yahoo Finance 或 Stooq：美股參考標的，例如 NVIDIA、SOX。

## 新聞資料

初期先用手動週報與半自動摘要，不急著買新聞 API。

可參考來源：

- 中央社
- 經濟日報
- 工商時報
- MoneyDJ
- Reuters
- Bloomberg
- CNBC
- Nikkei Asia

## 初期策略

先不要買資料。

第一版資料更新順序：

1. 股票主檔。
2. 每日價格。
3. 三大法人。
4. 融資融券。
5. 本益比 / 本淨比 / 殖利率。
6. 月營收。
7. 國際參考指標。
8. 新聞摘要。
